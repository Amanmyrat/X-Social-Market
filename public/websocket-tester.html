üìö Quick Start Guide

1. Configure Connection

Enter your WebSocket host, port, and app key. These are typically found in your .env file:

REVERB_HOST=localhost
REVERB_PORT=6001
REVERB_APP_KEY=your-app-key

2. Connect

Click the "Connect" button. Watch the debug log for connection status.

3. Subscribe to Channels

  • Public: my-channel
  • Private: private-my-channel
  • Presence: presence-my-channel

4. Listen for Events

Enter the event name (e.g., MessageSent) and subscribe. Watch the debug log for incoming messages.

5. Send Messages (Optional)

Send client events on private/presence channels. Event names must start with "client-".
